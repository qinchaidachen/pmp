# 文档更新日志 (Changelog)

本文档记录项目管理平台用户文档的版本更新历史和变更内容。

## 📋 格式说明

本文档采用 [Keep a Changelog](https://keepachangelog.com/) 格式，记录以下类型的变更：

- **新增** (Added) - 新增功能或内容
- **修改** (Changed) - 现有功能的变更
- **弃用** (Deprecated) - 即将移除的功能
- **移除** (Removed) - 已移除的功能
- **修复** (Fixed) - 问题修复
- **安全** (Security) - 安全相关更新
